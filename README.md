# Members Only

A simple members-only message board app built with Node.js, Express, MongoDB, Mongoose, Passport.js, EJS, and bcrypt.

## Features

- User signup and login
- Password hashing with bcrypt
- Authentication using Passport.js
- Members-only secret passcode system
- Create messages when logged in
- Hide author and date from non-members
- Admin users can delete messages
- MongoDB database with Mongoose models

## Tech Stack

- Node.js
- Express.js
- MongoDB
- Mongoose
- Passport.js
- EJS
- bcryptjs
- express-session
- express-validator
- method-override
